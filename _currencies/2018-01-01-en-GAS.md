	                      <h2>{{page.coin_a_name}}'s Technology</h2>
	                    </div>
	                      <div class="panel-content">
	                       NEO operates on a unique unified mechanism. Instead of proof of work or evidence-share, NEO uses Delegated Byzantine Fault Tolerance (dBFT). In dBFT, stakeholders in NEO vote for a group of employees for each block. The accounting nodes create blocks, and then the network must agree on the new block. This system requires fewer resources than other types of consensus, and it is also more capable. Using this system of elected accountants, NEO reaches 1,000 transactions per second and can theoretically reach up to 10,000 transactions per second with optimized optimizations. Compared to the current network capacity of Ethereum is 15 transactions per second, NEO has a clear advantage. This is especially true for the use of large-scale smart contracts.
